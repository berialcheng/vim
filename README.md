# Configuration for VIM

## How to use
### Add the symbolic link into your local configuarion like below
	ls -s .vim ~/.vim
	ls -s .vimrc ~/.vimrc

### Useful tips 
	nerd -- open the left file system tree explorer
	Ctrl-P -- open File lookup
